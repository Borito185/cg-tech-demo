# Tech Demo

## Features
### Minimal
- Multiple viewpoints
- Blinn-phong shading
- PBR shaders
- Material textures
- Normal mapping
- Environment mapping
- Smooth Bezier paths
- Hierarchical transformations
### Extra
- Shadows for multiple light sources
- Procedurally generated terrains
- Day-night system
- SSRT Screen Space Ray Tracing for specular reflections
- Water surfaces
